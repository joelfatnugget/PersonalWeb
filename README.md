### UI & Component Libraries
To achieve a unique and polished look, this project leverages a curated selection of UI kits:
- **[Skeleton](https://www.skeleton.dev/):** For cohesive design tokens and responsive layouts.
- **[Flowbite-Svelte](https://flowbite-svelte.com/):** For robust, interactive components based on Tailwind CSS.
- **[shadcn-svelte](https://shadcn-svelte.com/):** For highly customizable and accessible UI primitives.
- **[SMUI (Svelte Material UI)](https://sveltematerialui.com/):** For implementing Material Design principles where appropriate.

### Iconography
A rich set of icons to enhance visual communication:
- **[Iconify](https://iconify.design/):** Unified access to thousands of icon sets.
- **[Lucide Svelte](https://lucide.dev/):** Clean and consistent community-run icon library.
- **[Hugeicons Svelte](https://hugeicons.com/):** Premium-grade, stroke-based icons for a modern feel.
